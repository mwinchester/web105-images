Background Images 1

If you are going to use a background image, you may not want it to repeat.

Why would you use a background image instead of a normal image?
Because they are for visual effect only and have no meaning. They will be ignored by browsers for the blind.

Background Images 2

Please be very careful when using background images. It is very hard to read text on top of an image.

Image formats

JPG
Jpeg or jpg, this file format is best used for photographs. It is lossy, if you save it to a lower resolution you will permanantly lose information.
GIF
Used for drawings and clip art. Allows a transparent background. Can use simple animation. 8 bit color.
PNG
Supports 8 or 24 bit color, can use for photos or drawings, supports transparency and animation. Largest of the three file types.
SVG
Line drawings, Illustrator can create. Small file size.

Image Alignment

You can "float" an image to the left or to the right. This is different than simply aligning left or right. When you float an image, it breaks the styling order of the web page, so instead of listing one element right after another, the next element fill "float " up next to the floated element. When you want to go back to the normal order, you will have to clear the float.

Making images responsive

By setting the width of an image to a percent of the size of the containing div, and letting the height be auto, if you resize the page, the image will resize.
